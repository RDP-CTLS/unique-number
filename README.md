# unique-number

A tiny web page that hands out a unique number that is never repeated, starting at 4000. Whoever opens the page and takes a number gets the next one in a single shared sequence, on any device, with no login.

Everyone draws from the same running count, so the same number can never be given out twice, no matter how many people or devices are using it.
